# Portfolio Website

uzak0209のポートフォリオサイト

## 技術スタック

- Next.js 16
- TypeScript
- Tailwind CSS
- React 19

## セットアップ

```bash
npm install
```

## 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開きます。

## ビルド

```bash
npm run build
```

## 本番サーバーの起動

```bash
npm start
```

## デプロイ
